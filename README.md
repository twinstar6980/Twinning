# Twinning

个人工具箱。

## 模块组成

* `Kernel`
	
	内核，负责内部功能的实现。

* `Shell`
	
	外壳，提供基本的用户界面。

* `Script`
	
	脚本，控制工具的运行流程。

* `Assistant`
	
	助理，实现了额外的高级辅助功能。

* `Assistant Plus`
	
	助理+，面向 Windows 平台的特化版本。

## 平台支持

* 操作系统：`Windows 10+` 、`Linux ~` 、`Macintosh 11+` 、`Android 11+` 、`Iphone 15+` 。

* 处理器架构：`AMD64` 、`ARM64` 。
